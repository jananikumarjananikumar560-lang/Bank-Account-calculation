# Bank Account Management System

This is a simple Java project that simulates basic banking operations using Object-Oriented Programming (OOP) concepts.

## Features
- Create a bank account
- Deposit money
- Withdraw money
- Check account balance
- Simple console-based application

## Technologies Used
- Java
- Eclipse / VS Code

## Concepts Covered
- Classes and Objects
- Inheritance
- Method Overriding
- Encapsulation
- Constructors
- User Input using Scanner

## Output
The program allows users to perform basic banking operations such as deposit, withdrawal, and balance inquiry through a console interface.
